Detailed Description

Système d'exploitation du système RAID 5. Permet de lancer un enssemble de commandes, (ls [-l]; cat; rm; create; edit; load; store) afin de lister, lire, ecrire, editer, stocker, recuperer vers et depuis le systeme rAID 5. La commande quit permet de quiter.
